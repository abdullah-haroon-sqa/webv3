  1. Create a class called Point that has two data members: x and y-coordinates of the point. Provide a no argument and a 2-argument constructor. Provide separate get and set functions for each of the data members i.e. getX, getY, setX, setY. The getter functions should return the corresponding values to the calling function. Provide a display method to display the point in (x, y) format. Make appropriate functions const.
    Drive a class circle from the point class that has an additional data member: radius of the circle. The point from which this circle is derived represents the center of the circle. Provide a no-argument constructor to initialize the radius and center coordinates to 0. Provide a 2- argument constructor: one argument to initialize the radius of circle and the other argument to initialize the center of the circle (provide an object of point class int the second argument). Provide a 3-argument constructor that takes three floats to initialize the radius, x and y-coordinates of the circle. Provide setter and getter functions for radius of the circle. Provide two functions to determine the radius, area and circumference of the circle. Write main function to test the class.
